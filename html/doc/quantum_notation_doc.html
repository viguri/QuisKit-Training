Quantum Computing Mathematical Notation Guide

Dirac Notation (Bra-Ket)

Dirac notation is the standard notation for describing quantum states in quantum mechanics and quantum computing.

Key Elements

  • Ket notation \(|\psi\rangle\): Represents a quantum state vector
    • \(|0\rangle\): Represents the computational basis state 0
    • \(|1\rangle\): Represents the computational basis state 1
  • Bra notation \(\langle\psi|\): Represents the conjugate transpose of a ket
    • \(\langle0|\): Conjugate transpose of \(|0\rangle\)
    • \(\langle1|\): Conjugate transpose of \(|1\rangle\)

Common Quantum States

In our quantum circuits, we frequently use these states:

  • Basis States:
    • \(|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}\)
    • \(|1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}\)
  • Superposition States:
    • \(|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}\)
    • \(|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}\)

Quantum Gates in Matrix Form

Common quantum gates used in our circuits:

  • Hadamard Gate (H): \[H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}\] Creates superposition from basis states
  • Pauli-X Gate (NOT): \[X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\] Flips the state of a qubit
  • Phase Gate (S): \[S = \begin{pmatrix} 1 & 0 \\ 0 & i \end{pmatrix}\] Adds a phase of i to the \(|1\rangle\) state

Measurement

Measurement in the computational basis is denoted as:

  • For state \(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\):
  • Probability of measuring \(|0\rangle\): \(|\alpha|^2\)
  • Probability of measuring \(|1\rangle\): \(|\beta|^2\)
  • Where \(|\alpha|^2 + |\beta|^2 = 1\)

Example in Our Code


# Creating basis state |0⟩
zero_circuit = QuantumCircuit(1)  # |0⟩ by default

# Creating basis state |1⟩
one_circuit = QuantumCircuit(1)
one_circuit.x(0)  # Applies X gate to create |1⟩

# Creating superposition state |+⟩
plus_circuit = QuantumCircuit(1)
plus_circuit.h(0)  # Applies H gate to create |+⟩

Note: When we measure these states:

  • |0⟩ gives 0 with 100% probability
  • |1⟩ gives 1 with 100% probability
  • |+⟩ gives 0 or 1 with 50% probability each
  • |-⟩ gives 0 or 1 with 50% probability each

Guía de Notación Matemática en Computación Cuántica

Notación de Dirac (Bra-Ket)

La notación de Dirac es la notación estándar para describir estados cuánticos en mecánica cuántica y computación cuántica.

Elementos Clave

  • Notación Ket \(|\psi\rangle\): Representa un vector de estado cuántico
    • \(|0\rangle\): Representa el estado base computacional 0
    • \(|1\rangle\): Representa el estado base computacional 1
  • Notación Bra \(\langle\psi|\): Representa la transpuesta conjugada de un ket
    • \(\langle0|\): Transpuesta conjugada de \(|0\rangle\)
    • \(\langle1|\): Transpuesta conjugada de \(|1\rangle\)

Estados Cuánticos Comunes

En nuestros circuitos cuánticos, frecuentemente usamos estos estados:

  • Estados Base:
    • \(|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}\)
    • \(|1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}\)
  • Estados de Superposición:
    • \(|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}\)
    • \(|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}\)

Puertas Cuánticas en Forma Matricial

Puertas cuánticas comunes usadas en nuestros circuitos:

  • Puerta Hadamard (H): \[H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}\] Crea superposición desde estados base
  • Puerta Pauli-X (NOT): \[X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\] Invierte el estado de un qubit
  • Puerta de Fase (S): \[S = \begin{pmatrix} 1 & 0 \\ 0 & i \end{pmatrix}\] Añade una fase de i al estado \(|1\rangle\)

Medición

La medición en la base computacional se denota como:

  • Para el estado \(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\):
  • Probabilidad de medir \(|0\rangle\): \(|\alpha|^2\)
  • Probabilidad de medir \(|1\rangle\): \(|\beta|^2\)
  • Donde \(|\alpha|^2 + |\beta|^2 = 1\)

Ejemplo en Nuestro Código


# Crear estado base |0⟩
zero_circuit = QuantumCircuit(1)  # |0⟩ por defecto

# Crear estado base |1⟩
one_circuit = QuantumCircuit(1)
one_circuit.x(0)  # Aplica puerta X para crear |1⟩

# Crear estado de superposición |+⟩
plus_circuit = QuantumCircuit(1)
plus_circuit.h(0)  # Aplica puerta H para crear |+⟩

Nota: Cuando medimos estos estados:

  • |0⟩ da 0 con 100% de probabilidad
  • |1⟩ da 1 con 100% de probabilidad
  • |+⟩ da 0 o 1 con 50% de probabilidad cada uno
  • |-⟩ da 0 o 1 con 50% de probabilidad cada uno
