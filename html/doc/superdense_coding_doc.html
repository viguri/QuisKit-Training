Superdense Coding Protocol

Overview

Superdense coding is a quantum communication protocol that allows Alice to send two classical bits to Bob using one qubit, provided they share an entangled pair.

Initial Bell State

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Protocol Steps

1. Bell Pair Creation

Initial state:

$$|0\rangle \otimes |0\rangle \xrightarrow{H \otimes I} \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) \otimes |0\rangle$$ $$\xrightarrow{CNOT} \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

2. Alice's Encoding

Four possible messages encoded by Alice:

  • 00: $|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$ (No operation)
  • 01: $|\Psi^+\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)$ (X gate)
  • 10: $|\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)$ (Z gate)
  • 11: $|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$ (ZX gates)

3. Bob's Decoding

Bob's operations:

$$|\psi\rangle \xrightarrow{CNOT} |\psi'\rangle \xrightarrow{H \otimes I} |message\rangle$$

Security Analysis

Eve's Limitations

Eve cannot intercept the message because:

  1. No-cloning theorem prevents copying quantum states
  2. Measurement disturbs the quantum state
  3. Entanglement provides quantum security

Mathematical Security

If Eve tries to measure the qubit in transit:

$$|\psi\rangle \xrightarrow{\text{measure}} \rho_{mixed} = \frac{1}{2}|0\rangle\langle 0| + \frac{1}{2}|1\rangle\langle 1|$$

This destroys the entanglement and thus the message.

Implementation Results

Circuit Implementation

Bell pair creation:

$$|00\rangle \xrightarrow{H_1} \frac{1}{\sqrt{2}}(|00\rangle + |10\rangle) \xrightarrow{CNOT} \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Message: 00

Message: 11

Quantum Information Theory

Information Content

Classical channel capacity: 1 bit per qubit

Superdense coding capacity: 2 bits per qubit (with entanglement)

Holevo bound for single qubit without entanglement:

$$\chi \leq 1 \text{ bit}$$

With entanglement assistance:

$$\chi \leq 2 \text{ bits}$$

Protocolo de Codificación Superdensa

Descripción General

La codificación superdensa es un protocolo de comunicación cuántica que permite a Alice enviar dos bits clásicos a Bob usando un qubit, siempre que compartan un par entrelazado.

Estado Bell Inicial

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Pasos del Protocolo

1. Creación del Par Bell

Estado inicial:

$$|0\rangle \otimes |0\rangle \xrightarrow{H \otimes I} \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) \otimes |0\rangle$$ $$\xrightarrow{CNOT} \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

2. Codificación de Alice

Cuatro posibles mensajes codificados por Alice:

  • 00: $|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$ (Sin operación)
  • 01: $|\Psi^+\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)$ (Puerta X)
  • 10: $|\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)$ (Puerta Z)
  • 11: $|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$ (Puertas ZX)

3. Decodificación de Bob

Operaciones de Bob:

$$|\psi\rangle \xrightarrow{CNOT} |\psi'\rangle \xrightarrow{H \otimes I} |message\rangle$$

Análisis de Seguridad

Limitaciones de Eve

Eve no puede interceptar el mensaje porque:

  1. El teorema de no-clonación impide copiar estados cuánticos
  2. La medición perturba el estado cuántico
  3. El entrelazamiento proporciona seguridad cuántica

Seguridad Matemática

Si Eve intenta medir el qubit en tránsito:

$$|\psi\rangle \xrightarrow{\text{measure}} \rho_{mixed} = \frac{1}{2}|0\rangle\langle 0| + \frac{1}{2}|1\rangle\langle 1|$$

Esto destruye el entrelazamiento y por lo tanto el mensaje.

Resultados de Implementación

Implementación del Circuito

Creación del par Bell:

$$|00\rangle \xrightarrow{H_1} \frac{1}{\sqrt{2}}(|00\rangle + |10\rangle) \xrightarrow{CNOT} \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Mensaje: 00

Mensaje: 11

Teoría de Información Cuántica

Contenido de Información

Capacidad del canal clásico: 1 bit por qubit

Capacidad de codificación superdensa: 2 bits por qubit (con entrelazamiento)

Límite de Holevo para un qubit sin entrelazamiento:

$$\chi \leq 1 \text{ bit}$$

Con asistencia de entrelazamiento:

$$\chi \leq 2 \text{ bits}$$
