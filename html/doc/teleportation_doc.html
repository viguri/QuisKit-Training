Quantum Teleportation Protocol: Mathematical Foundation

Initial State Setup

System Components

The quantum teleportation circuit uses three qubits:

  • Qubit 0: The message qubit (Alice's qubit to be teleported)
  • Qubit 1: Alice's entangled qubit
  • Qubit 2: Bob's entangled qubit

Step 1: Message Qubit Preparation

In our implementation, we prepare the message qubit in state |1⟩:

Initial state of qubit 0:

$$|q_0\rangle = |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$

Code implementation:

circuit.x(0) # Set qubit 0 to |1⟩

Step 2: Creating the Bell State

We create a Bell state between qubits 1 and 2 using:

  1. Hadamard gate on qubit 1
  2. CNOT gate between qubits 1 and 2

1. After Hadamard on qubit 1:

$$|q_1\rangle = H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}$$

2. After CNOT between qubits 1 and 2:

$$|\beta_{00}\rangle_{12} = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Complete Initial State

The total state of the system is:

$$|\psi\rangle = |1\rangle_0 \otimes \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)_{12}$$ $$= \frac{1}{\sqrt{2}}(|100\rangle + |111\rangle)$$

Protocolo de Teleportación Cuántica: Fundamento Matemático

Configuración del Estado Inicial

Componentes del Sistema

El circuito de teleportación cuántica utiliza tres qubits:

  • Qubit 0: El qubit mensaje (qubit de Alice a teleportar)
  • Qubit 1: Qubit entrelazado de Alice
  • Qubit 2: Qubit entrelazado de Bob

Paso 1: Preparación del Qubit Mensaje

En nuestra implementación, preparamos el qubit mensaje en el estado |1⟩:

Estado inicial del qubit 0:

$$|q_0\rangle = |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$

Implementación en código:

circuit.x(0) # Establecer qubit 0 a |1⟩

Paso 2: Creación del Estado Bell

Creamos un estado Bell entre los qubits 1 y 2 usando:

  1. Puerta Hadamard en el qubit 1
  2. Puerta CNOT entre los qubits 1 y 2

1. Después de Hadamard en el qubit 1:

$$|q_1\rangle = H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}$$

2. Después de CNOT entre qubits 1 y 2:

$$|\beta_{00}\rangle_{12} = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Estado Inicial Completo

El estado total del sistema es:

$$|\psi\rangle = |1\rangle_0 \otimes \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)_{12}$$ $$= \frac{1}{\sqrt{2}}(|100\rangle + |111\rangle)$$
